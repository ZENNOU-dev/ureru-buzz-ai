# 台本作成

## INPUT
- 構成案（パート順序と各パートの目的・素材）

## OUTPUT
- 台本（具体的な言葉）

---

## 執筆の原則

### 毎秒の脳内設計
- **テンプレを埋めるな。視聴者の脳内を1秒ずつ設計しろ**
- 枠に当てはめるのではなく「今何を思っているか？次に何を思わせたいか？」を毎秒考える

### 言葉の原則
- 言葉の強さ = **一語から伝わる情報量。** ユーザーの脳の記憶ネットワークを活用する
- 動画広告は音声で伝える → **口に出して読んだ時の自然さ**
- 同じことを伝えるなら**できる限り短い文章**

### フックの言葉
- 出来事を描写する。感情は書かない。感情は視聴者が勝手に生成する
  - NG: 「墓まで持ってく」「心の中で土下座」→ 気持ち
  - OK: 「コンビニで免許証出すたびに二度見される」→ 出来事
- 「誰に言われるか」の解像度。20代女性: すきぴ/初対面の男/同性の嫉妬 ○ / 母親/歯科衛生士 ×
- 自慢×自虐のバランス。LLMは自慢に寄りがち→意識的に下げを入れる
- N1の脳内マップ上の言葉で入る

### 認知転換の言葉
- REVEALの構造。驚き→理由の順
- 商品名が出る前にコンテンツとして認知転換を完了させる

## セルフチェック

台本を書いた後、提出前に必ず確認:

- [ ] 毎秒の視聴者の脳内状態をシミュレーションしたか？
- [ ] N1の言葉で語っているか？商品カテゴリーで入っていないか？
- [ ] 商品名が出るタイミングは適切か？
- [ ] 教育の前に「知りたい」状態を作っているか？
- [ ] 認知転換はREVEALか？講義になっていないか？
- [ ] テンプレ埋めになっていないか？
- [ ] 緊張→解決を使えているか？
- [ ] フックに詰められるものを温存していないか？
- [ ] 口に出して読んで自然か？
