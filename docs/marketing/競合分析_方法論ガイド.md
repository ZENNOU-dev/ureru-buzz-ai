# 競合分析

## INPUT
- リサーチデータ（3C）

## OUTPUT
- 競合のポジション・訴求・CR傾向
- ユーザーの脳内マップ上の空きポジション

---

## 競合の3層

### 直接競合
同一マーケット。例: ローコストなら歯科矯正各社。
- 5つの価値軸で整理（機能/情緒/金銭/信頼/体制）

### 間接競合
類似の便益を提供する別マーケット。例: 骨切り、小顔整体。
- N1の脳内では同じ「解決策」カテゴリーに入っている

### 広告マーケット
直接・間接競合のCRがどんな訴求をしているか。
- WHO（N1）/ WHAT（セールスの核）/ HOW（コンテンツ）
- 出稿量が多いCR傾向
- → 詳細なCR分析は `CR分析.md` で実行

## 想定競合
直接競合 + 代替手段 + **無行動（何もしない）** ← 最大の競合
