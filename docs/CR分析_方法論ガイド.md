# CR分析

## INPUT
- 動画CR（自社 or 競合）

## OUTPUT
- CRの構造・セールス評価・N1特定・戦略の逆算

---

## 基本姿勢
- テキストではなく**人間を分析する**
- 「何が書いてあるか」ではなく「このコンテンツでCVする人間は誰か」

## 分析方法（技術）
- 手動スクショではなく**パイプラインで処理**（ffmpeg scene detect → OCR → Gemini transcription）
- フック分析は実テキスト（S0テロップ+S1テロップ+台本1文目）で。CR名ではない
- 台本全文を使わない（ボディが混入する）

---

## ボディ分析ワークフロー

### Step 1: 構造分解
フック / 教育 / 信頼構築 / オファー / CTA に分ける

### Step 2: CVする人のプロファイリング
N1の7項目+ファネルで定義

### Step 3: ボディの各パートがN1のどこに刺さっているか
- 教育パート → 「知らないこと」に対応
- 信頼構築 → 「選択基準」に対応
- オファー → 「行動ハードル」に対応
- 教育あり → 潜在〜準顕在層 / 教育なし → 顕在〜比較検討層

### Step 3.5: 自分ごと化装置の特定
一般論→「これ私のことだ」に変換するトリガーは何か

## セールス設計図での評価

**知覚価値（6要素）を◎○△❌で評価:**
便益の正しい解釈 / ポジショニング / 唯一性 / 自分ごと化 / 証拠 / 便益の重なり

**知覚コスト（7要素）を◎○△❌で評価:**
金銭 / 金銭の正当性 / リスク / 比較 / 手間 / 心理 / 時間

---

## コンテンツ分析の型

1. **セールスの核（認知転換）を特定する**
2. **直前にどんな認知状態が必要か**
3. **コンテンツがその認知状態を作れているか**

全ての要素がセールスの核に向かっている = 良い構成

---

## 逆算分析

売れてるCR = 市場の答え合わせ。逆順でたどる:

| 段階 | 問い |
|------|------|
| 台本 | 何と言っているか？どんな素材？ |
| 構成 | どのパートが入っていてどれがないか？ |
| 企画 | ファネルは？表現手法は？フック型は？ |
| 訴求 | WHO / WHAT(コンセプト) / WHY |
| リサーチ | 必要だったマーケット知識は？ |

---

## 4つの壁での評価

| 壁 | 担当 | KPI |
|----|------|-----|
| 1a. 見始めない | フック（0.5秒） | 3s視聴率 |
| 1b. 見続けない | つなぎ / 教育の引き | 25%再生率 |
| 2. 信じない | 教育+信頼構築 | 完視率・CTR |
| 3. 行動しない | オファー+CTA | LPCVR・CVR |
